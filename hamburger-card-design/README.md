# 🍔 Hamburger Product Card UI

This is a hamburger product card UI project I created while learning web development.

Although I am still a beginner developer in the learning process, I am studying hard through books, online courses, and hands-on practice.

Based on concepts I learned just a few days ago, I designed and built this product card component inspired by the menu screens of fast-food brands.

![Hamburger Product Card UI](hamburger_card.png)


## 🍔 Features

* Product card layout design
* NEW / BEST badges
* Discount rate and sale price display
* Hover effect that enlarges the product image
* Add to Cart button section
* Responsive design optimized for mobile devices

## 🛠 Technologies Used

* HTML5
* CSS3
